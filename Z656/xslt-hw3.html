Using this XML as your source, create an XSLT transformation that turns the information into separate HTML web pages for each journal article using result-document.

Your XSLT should include the following:

  1. Treatment of the different text styles (bold, italics, etc.)
  2. Some sort of styling for positional elements (this thing should be on the right side, etc.)
  3. Metadata at the beginning of each article.

Make your HTML files link to a CSS stylesheet developed specifically for these articles and host them on GitHub in a subdirectory containing all articles and your XSLT.

Your CSS should:

  1. Include an uploaded font
  2. Make your articles look old-timey by mimicking the original appearance of them (see the PDFs).

Please submit a link to this subdirectory in Canvas.
