Using this XML as your source, create an XSLT transformation that turns the information into an HTML web page. Be sure to include some descriptive info and data headings so the page makes sense.

Your XSLT should include the following:

  1. Introductory general information about the game
  2. Two tables that convey relevant box score data for each team
  3. A variable that contains a concatenation of relevant game information, to be used more than once in your HTML
  4. Sorting based on some metadata about individual players: batting average, last name, jersey number, etc.

Make your HTML file link to the CSS stylesheet you developed for your GitHub Pages site and host the HTML and XSLT on GitHub in a subdirectory. Feel free to make adjustments to your GitHub Pages CSS as you see fit. Please sumbit a link to this subdirectory in Canvas.
