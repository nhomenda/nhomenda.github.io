Using your selected XML standard, take 15 minutes to prepare a 3 minute presentation to the class about it, describing the standard by answering these questions:

  • What’s its homepage?
  • Who created it and when?
  • What’s its governing body now, or, who maintains it?
  • When was it last updated?
  • What is it used to primarily encode?
  • How human-readable is it: do the element and attribute names make any sense?
  • Who uses this standard?
  • What’s the root element?
  • What are the main elements you’d see just below the root element?
  • Would you use this in a digital publishing system? For publishing what?

In addition, please show a real-world example of one of these XML files and/or a project or system that uses them.
