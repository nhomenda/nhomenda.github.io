Here are basic XPath tenets:

  1. Most XPaths begin with /
  2. /root/parent/child selects all children at that level
  3. Predicates
    1. /root/parent/child[2] selects second child
    2. /root/parent/child[@name] selects all children with an @name
    3. /root/parent/child[@name="Fabio"] selects all children with @name="Fabio"
  4. Shortcuts
    1. * is a wildcard
    2. //element looks for element anywhere, is slow
  5. Functions
    1. count(something) gives a number of somethings
    2. substring-before(something, thing) returns "some"
    3. substring-after(something, some) returns "thing"
